测试并发布

  • 新建发布账户在RubyGems.org。

  • 新建代码远程仓库。

  • 更新 fastlane-plugin-[plugin_name].gemspec 文件 的 spec.homepage 指向远程仓库的 repo。

  • 提交代码到远程仓库。
